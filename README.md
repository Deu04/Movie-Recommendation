# 🎬 Movie Recommendation System

## 📖 Project Overview

The Movie Recommendation System is a machine learning-based application that recommends movies similar to a user's selected movie. The system uses content-based filtering techniques to analyze movie similarities and generate personalized recommendations.

Built using Python and Streamlit, the application provides an interactive user interface where users can select a movie from a dropdown menu and instantly receive recommended movies along with their posters fetched from The Movie Database (TMDB) API.

This project demonstrates practical implementation of recommendation systems, data processing, API integration, and web application development.

---

## 🚀 Features

* Movie recommendation based on similarity scores
* Interactive Streamlit web interface
* Real-time movie poster fetching using TMDB API
* Fast recommendation generation
* User-friendly movie selection dropdown
* Displays top recommended movies with posters

## 📊 Project Workflow

1. Load movie dataset and similarity matrix
2. User selects a movie from the dropdown menu
3. System calculates the most similar movies
4. Fetch movie posters using TMDB API
5. Display recommended movies with posters
